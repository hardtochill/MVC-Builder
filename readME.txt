该程序是一个SpringBoot架构代码生成器
实现功能：解析数据库自动生成Controller、Service、DAO三层架构与CRUD代码。 
读取并解析YML配置文件，通过JDBC连接数据库并完成对表字段、索引、注释等表信息的准确解析。
根据解析后的表信息，自动生成Controller层、Service层、DAO层基本的CRUD代码，遵循SpringBoot分层架构，具备良好可读性和可维护性。
生成MapperXML文件，支持依据联合索引对数据进行查改删操作，支持批量插入、分页查询等基本功能。
自动依据表注释与业务逻辑为代码生成注释，包含功能、参数等基本信息，方便开发人员快速理解代码。 

1.需手动为目标项目导入pagehelper依赖
2.需手动在配置文件中开启驼峰命名
3.dto、entity、vo虽然生成出来，但用户需根据需要更改对应信息
4.PageQueryDTO对象直接继承自对应DTO类，在此基础上添加了分页查询的属性。开发者可根据需求选择不继承DTO类，直接编写对应分页查询的属性。
